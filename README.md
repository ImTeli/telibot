# Telibot, a discord bot!

## Purpose

* Make discord interactions more fun and provide some functionality.

## Features

* Play a sound whenever someone that is not connected to any voice channel joins a voice channel.
* Play a specific sound by naming that sound.
* TTS (Text to speech).

## Future Features

* Some moderation funcionality (like supect link removal) and SPAM detection.
